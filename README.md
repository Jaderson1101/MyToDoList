# MinhaWebTarefas

Projeto web simples desenvolvido com ASP.NET Core e Razor Pages.

## Funcionalidades (em progresso)

- [ ] Cadastro de tarefas
- [ ] Listagem de tarefas
- [ ] Edição de tarefas
- [ ] Exclusão de tarefas
- [ ] Filtros por status e data
- [ ] Autenticação de usuários

## Tecnologias

- .NET 8.0
- ASP.NET Core
- Razor Pages
- Entity Framework Core
- SQLite (planejado)

## Objetivo

Projeto pessoal para praticar desenvolvimento web com C# e ASP.NET Core.
